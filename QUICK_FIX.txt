  🚨 إصلاح Termux السريع - TERMUX QUICK FIX
═══════════════════════════════════════════════════════════════

⚠️ المشكلة: pkg و chmod غير موجودين في Termux

✅ الحل السريع (انسخ والصق):

───────────────────────────────────────────────────────────────
الطريقة 1: استخدام apt (جرب هذا أولاً)
───────────────────────────────────────────────────────────────

apt update && apt install -y termux-tools coreutils && pkg update && pkg install -y nodejs git python

───────────────────────────────────────────────────────────────
الطريقة 2: تغيير المستودع
───────────────────────────────────────────────────────────────

termux-change-repo

ثم اختر مستودع مختلف، ثم:

apt update && apt install -y termux-tools coreutils

───────────────────────────────────────────────────────────────
الطريقة 3: تشغيل سكريبت الإصلاح
───────────────────────────────────────────────────────────────

انتقل لمجلد المشروع:

cd ~/claude-code

إذا لم يكن موجود، استنسخه:

cd ~ && git clone https://github.com/mhdessouky-creator/claude-code.git && cd claude-code

ثم شغل سكريبت الإصلاح (بدون chmod):

sh fix-termux-now.sh

───────────────────────────────────────────────────────────────
الطريقة 4: إعادة تثبيت Termux (الحل الأخير)
───────────────────────────────────────────────────────────────

1. Settings → Apps → Termux → Storage → Clear Data
2. أو: امسح التطبيق وأعد تثبيته من Google Play / F-Droid
3. افتح Termux من جديد
4. شغل:

pkg update && pkg upgrade -y

pkg install -y termux-tools coreutils nodejs git python

───────────────────────────────────────────────────────────────
بعد الإصلاح: استعادة المشروع
───────────────────────────────────────────────────────────────

cd ~
git clone https://github.com/mhdessouky-creator/claude-code.git
cd claude-code
sh termux-recovery.sh

───────────────────────────────────────────────────────────────
للتحقق من نجاح الإصلاح:
───────────────────────────────────────────────────────────────

which pkg       # يجب أن يظهر: /data/data/com.termux/files/usr/bin/pkg
which chmod     # يجب أن يظهر: /data/data/com.termux/files/usr/bin/chmod
node --version  # يجب أن يظهر رقم الإصدار
git --version   # يجب أن يظهر رقم الإصدار

═══════════════════════════════════════════════════════════════

💡 نصائح:

1. لا تشغل أوامر curl | sh مباشرة
2. استخدم termux-change-repo إذا كانت التحديثات بطيئة
3. احتفظ بنسخة احتياطية من .env
